# Development Workflow

## Principles
1.  **User-Centric Design:** Prioritize the ease of use for non-technical users (therapists).
2.  **Iterative Development:** Build, test, and refine features in small increments.
3.  **Code Quality:** Maintain clean, semantic HTML and modular CSS/JS.

## Process
1.  **Plan:** Define the scope of work in a Track.
2.  **Implement:** specific features or fixes.
3.  **Verify:** Test the generator's output and the UI itself.
4.  **Review:** Ensure alignment with product goals.
